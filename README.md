# asmfinance-web

Public landing site for **ASM Investments** — `https://asmfinance.in` (custom domain on GitHub Pages).

Sister repo: [`asmfinance`](../asmfinance) (Go API + admin SPA, hosted on Hostinger VPS at `api.asmfinance.tech` and `admin.asmfinance.tech`).

## What lives here

Marketing site only. Static. No business logic. The lead form POSTs to `https://api.asmfinance.tech/v1/leads` (CORS-allowed).

## Stack (decided in ADRs)

- **Astro** — content-first, static output, React islands for the contact form
- **React + Tailwind** — for the form island and any interactive widgets
- **GitHub Pages** — hosting, free TLS via custom domain
- **GitHub Actions** — build + deploy on push to `main`

## Getting started

This repo currently contains **planning docs only**. Code is to be generated by Claude Code following `docs/01-claude-code-prompt.md`.

1. Read `docs/00-START-HERE.md`.
2. Open Claude Code at this repo root.
3. Paste the contents of `docs/01-claude-code-prompt.md` as the initial message.
4. Review the scaffold, then iterate.
